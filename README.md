## Ubuntu での設定

### 仮想デスクトップ操作

設定場所: 設定 > キーボード > キーボードショートカットの表示とカスタマイズ

- 左右のワークスペース移動
  - カテゴリ: ナビゲーション
    - 左のワークスペースへ切り替える: Ctrl + ←
    - 右のワークスペースへ切り替える: Ctrl + →
- 仮想デスクトップ一覧 (Mission Control風)
  - カテゴリ: システム (または検索で「アクティビティ」)
    - アクティビティ画面を表示する: Ctrl + ↑

キーボード環境: HHKB (JIS)
